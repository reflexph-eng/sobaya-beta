# Dossier des encarts publicitaires SOBAYA

Dépose ici les images de tes espaces publicitaires.
Elles sont automatiquement accessibles sans déploiement supplémentaire.

## Utilisation

1. Copie ton image dans ce dossier (ex: `banniere-top.jpg`)
2. Dans /admin/pub, renseigne l'URL : `/ads/banniere-top.jpg`
3. Active l'emplacement — c'est en ligne immédiatement.

## Formats recommandés

| Emplacement | Fichier suggéré | Dimensions |
|---|---|---|
| Bannière haute | banner-top.jpg | 1200 × 120 px |
| Colonne gauche 1 | sidebar-left-1.jpg | 300 × 250 px |
| Colonne gauche 2 | sidebar-left-2.jpg | 300 × 250 px |
| Colonne droite | sidebar-right.jpg | 300 × 600 px |

## Formats acceptés
JPG, PNG, WebP — poids recommandé : moins de 200 Ko par image
pour un chargement rapide sur mobile.
