## Comment activer une photo

1. Dépose ton fichier JPG/PNG/WebP dans ce dossier
2. Va sur /admin/about dans le dashboard SOBAYA
3. Saisis le chemin : /about/fondateur.jpg
4. Enregistre — la photo s'affiche immédiatement
